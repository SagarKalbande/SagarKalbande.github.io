Description

Rescue Path-Finder is a Desktop Simulation Application where the user can place a solider at a location, and a victim at another location. After the simulation is run, the solider would follow the shortest path to get to the victim. This application was made using Unreal Engine. The next milestones in this application would be to give weights to different paths and make the soldier traverse accordingly.
